**Task Manager** is a minimalist task management application designed to help
users organize their daily tasks and stay productive. This application allows
users to create, edit, and delete tasks, providing a straightforward interface
for efficient task management.

### Key Features <a name="key-features"></a>

- **Edit existing tasks to update their details or mark them as done.**
- **Delete tasks you no longer need.**
